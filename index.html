About

Tomasz Jaworski

I have been programming for nine years, and my passion for game development was sparked at the very beginning of this journey. I learned mainly through online courses, which allowed me to independently explore various technologies and tools. I participated in numerous competitions and game jams, which not only expanded my skills but also gave me the opportunity to collaborate with other enthusiasts.

Over the past 3.5 years, I have had the chance to grow professionally and contribute to the development of mobile games. At the same time, I have been working on chess engines. My best project, Jackal, plays at a superhuman level (over 3000 ELO) and leverages machine learning. I am particularly fascinated by creating AI for game opponents, combining my interests in programming and artificial intelligence.

I am committed to continuously improving my skills. I stay up-to-date with the latest advancements in the gaming world and development processes. My dream is to one day create a world-class game that combines innovation with the passion I pour into every project.

ApocalypseZOZO

I made this game with my friend in just two days for the GMTK Game Jam 2020. The theme was "Out of Control." The map is randomly generated using premade prefabs. The main goal of the game is to stop the city from turning into a zombie town. We ended up spending a lot of time on the technical stuff, so we didn’t really get a chance to polish the gameplay. Even so, I’m proud of what we managed to put together. Hope you enjoy it!

Jackal

This is one of my recent projects—a chess engine written in Rust. It uses MCTS (Monte Carlo Tree Search) and neural networks to play at a superhuman level, with a strength of over 3000 Elo. In this engine, I’m experimenting with ways to tweak MCTS and the neural network to make its playstyle more aggressive.

Javelin

This is my first project in Rust. I decided to try creating a chess engine using the less common MCTS (Monte Carlo Tree Search) technique. The result is an engine that plays at a level of around 2800 Elo. Working on it also gave me a chance to learn more about training and implementing neural networks. I haven’t set up a Lichess bot for it yet, but you can check out the code on my GitHub.

One Room?

I made this game with my friend in two days for the Indie Tales Game Jam 2021. The theme was "One Room." We thought it’d be cool to use an orthographic camera to show multiple copies of the same room. The goal is to spot and click on as many rooms with a missing object as you can before the timer runs out. Hope you have fun!

Screw These Loops

I made this game in two days with my friends for Ludum Dare 47. The theme was "Stuck in a Loop," so I wanted to create a game where the player has to break things to progress. It’s a short story-driven puzzle game about a character trying to escape from a code loop. Hope you enjoy it!

Neural Network Library

This is a very simple multilayer perceptron based on the JavaScript Neural library. I also made a Unity implementation of it, but I didn’t see much point in including it here. I’ve added my matrix library to the files as well.

Random Land Generator

This is a random land generator I made using the Fast Noise library. It was my first attempt at something like this, and I’m pretty happy with how it turned out. I also created a water shader for the project, which you can see in the image. Unfortunately, I no longer have the code for this project.

Sorting Library

This is a sorting library I made in C#. It includes Quick Sort, Bubble Sort, and Insertion Sort. I created it to get more hands-on experience with these sorting algorithms and to practice writing cleaner code. It was a few years ago, but looking back, I still think the code wasn’t too bad. Though, in hindsight, I might have gone a little overboard with the comments.
